## 서비스 소개
가상의 햄버거 프렌차이즈 업체(본사&지점) 내부관리시스템 및 비대면 웹 주문 키오스크
본사와 지점을 위한 맞춤형 매출 차트! (매출관리 및 제품 선호도 파악 용이)

1. 본사,지점 : 모든 지점 혹은 각 지점의 매출상황과 메뉴/재고/발주의 원활한 관리
2. 사용자 : 비대면으로 쉽고 간편하게 주문 대기시간 단축
